# Tomty

Simple [Tomtit](https://github.com/melezhik/Tomtit) based test runner

# Install

    zef install Tomty


# Usage


    tom --edit test-one

    bash "echo Hello World"

    tom --edit test-two

    bash "echo Upps && exit 1";


    tomty # Runs all test-* scenarios and make report
